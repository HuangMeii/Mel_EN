Tham khảo nhanh

Thì cơ bản
Present Simple: I work / He works
Diễn tả thói quen, sự thật hiển nhiên
Present Continuous: I am working
Diễn tả hành động đang xảy ra
Present Perfect: I have worked
Diễn tả kết quả của hành động trong quá khứ
Past Simple: I worked
Diễn tả hành động đã hoàn thành trong quá khứ
Câu điều kiện
Type 0: If + present, present
If you heat water, it boils.
Type 1: If + present, will + V
If it rains, I will stay home.
Type 2: If + past, would + V
If I had money, I would travel.
Type 3: If + past perfect, would have + V3
If I had studied, I would have passed.
